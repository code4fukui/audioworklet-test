# audioworklet-test

このリポジトリには、ブラウザ内でカスタムオーディオ処理ノードを作成するための AudioWorklet API の使用例が含まれています。

## デモ
デモは https://code4fukui.github.io/audioworklet-test/ でご覧いただけます。

## 機能
- シンプルなビープ音
- 複数のビープ音
- シンプルなノイズ音
- JSNES の PAPU.js を使用した PAPU 音源テスト
- IchigoJam MML の再生
- サイン波の生成
- CC0 音声サンプルを使用した風の音
- サンプリング音源 - Code for FUKUI の sound-cc0 を使用したクラッカー音

## 必要条件
このプロジェクトを実行するには、AudioWorklet API をサポートするモダンなウェブブラウザが必要です。

## 使い方
サンプルを実行するには、対応する HTML ファイルをウェブブラウザで開いてください。
